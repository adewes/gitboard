Github + Kanban = Gitboard

Gitboard is a simple and intuitive Kanban board for your Github issues. Easily manage to-dos, milestones and projects and keep up-to-date on your progress. Gitboard is open-source and runs securely inside your browser.

Questions & Answers

Is Gitboard commercial?

No! Gitboard is open-source software and licensed under a Affero General Public License. That means you can download, modify and republish Gitboard without charge, as long as you conform to the requirements of the license.

Does Gitboard require a server to run on?

No! Gitboard is a single-page application that runs entirely in your browser. All communication with the Github API is done directly, without use of a third-party server.

Is Gitboard secure?

Yes! Gitboard communicates directly with the Github API without using a third-party. You access credentials therefore stay on your computer. Gitboard does not store any user information or passwords in plaintext and will erase all data from your browser after you log out.

How can I contribute to Github?

As Gitboard is open-source, contributions are highly welcome! Our Github page contains all the information you need to get started. If you get stuck, feel free to send us an e-mail (see Github).

Which technologies is Gitboard built with?

Gitboard is built with React.js, Bootstrap (+Material Design) and various JS libraries such as jQuery, Moments.js and Marked.js. It uses SASS for CSS, Bower for dependency management and Make for building.
